## 📌 Overview
This project is a Machine Learning-based web application that predicts the likelihood of diabetes using user-provided health parameters.  
It is deployed using Streamlit, allowing real-time predictions through an interactive interface.

---

## 🚀 Features
- Real-time diabetes prediction
- Interactive and user-friendly UI
- Data preprocessing and feature handling
- Machine Learning model integration
- Instant result generation

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Libraries:** Scikit-learn, Pandas, NumPy  
- **Deployment:** Streamlit  
- **Tools:** GitHub  

---

## ⚙️ How It Works
1. User enters health parameters (e.g., glucose, BMI, age, etc.)
2. Data is preprocessed and passed to the trained ML model
3. The model predicts whether the person is diabetic or not
4. Result is displayed instantly on the web interface
